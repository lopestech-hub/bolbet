# Design System - bolbet

## Vision

**Direction:** Precision & Density
**Foundation:** Dark Mode (Cool Zinc/Slate)
**Depth:** Borders-only (No shadows for technical focus)
**Typography:** Inter (UI) + Geist Mono / JetBrains Mono (Data/Numbers)
**Interaction:** Fast, 150ms transitions, no bounce.

## Grid & Spacing

- Base: 4px
- Component Gap: 8px
- Page Padding: 16px

## Components

### Card (Trader View)

- Border: 1px solid rgba(255, 255, 255, 0.05)
- Background: #09090b (Zinc 950)
- Padding: 12px
- Radius: 4px

### Status Badges

- LIVE: Green background/text, subtle glow.
- FINISHED: Gray muted.

### Data Grid

- Header: Zinc 500, 11px, Uppercase, Tracking 0.05em.
- Rows: Zinc 100, 13px, Monospace for numbers.

## Colors

- Primary (BG): #020617 (Slate 950)
- Surface: #0f172a (Slate 900)
- Accent: #3b82f6 (Blue 500)
- Positive: #10b981 (Emerald 500)
- Danger: #ef4444 (Red 500)
